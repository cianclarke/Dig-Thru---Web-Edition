Dig Thru1.0a - developed by

Dig Thru is a tool to discover Antipodes" (see Wikipedia). It is implemented in AJAX with the JQuery library.
The calculation is simple:
   var endLat = start[0] * -1;
   var endLong = (start[1]>0) ? start[1]-180 : start[1] + 180;

The moving map implementation comes from Chris Heillman's awesome map.js.
